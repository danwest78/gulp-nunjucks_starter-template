# Requirements

- NodeJS / NPM / Bower

# Setup

This requires newer versions of NPM so Update NPM first

````bash
sudo npm install -g npm

# OR #

sudo npm update -g npm
````

Install Global Dependencies

````bash
sudo npm install -g gulp
````

Install local Dependenies

````bash
npm install
bower install
````

# Tasks

````bash
# clean and build everything
gulp

# watch & build
gulp watch

# cleanup
gulp clean
